### What you type

~~~
Inline code span: `let x = 5`

Code block:
```
def f(x):
   return x+1
```

Syntax highlighting:
```python
def fib(n):
    # TODO: base case
    return fib(n-1) + fib(n-2)
```
~~~

!!! tip ""

    You can also use `~~~` to start code blocks, or just indent the code 4 or more
    spaces.
